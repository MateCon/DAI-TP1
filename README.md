## Setup

1. Clonar Repo
2. Generar la base de datos con el script en ./db/db.sql con mssql
3. Agregar .env con las variables de la base de datos para ./src/utils/database.ts
4. Iniciar el proyecto (con node y npm instalados)

```shell
npm i
npm run dev
```
